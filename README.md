------------------------------------
Project Overview
------------------------------------
This project implements a machine learning pipeline for phishing URL detection
using lexical and structural features. The system supports model training,
feature selection, model comparison, and real-time URL classification via a
command-line interface.

------------------------------------
Requirements
------------------------------------
Python 3.10 or later

Install required packages:
pip install -r requirements.txt

------------------------------------
Running the Project
------------------------------------
Navigate to the src directory and run:

python main.py

This will launch the command-line interface where you can:
- Load dataset
- Extract features
- Train models
- Run experiments
- Classify URLs

------------------------------------
Project Structure
------------------------------------
src/     - Source code
data/    - Dataset files
requirements.txt - Python dependencies
README.txt - Instructions

------------------------------------
Dataset
------------------------------------
The dataset used is a labelled phishing URL dataset in CSV format located in
the data folder.
